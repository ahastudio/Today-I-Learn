## 1. python 스터디
- 러닛에서 2달간 진행한 python 스터디를 진행함
- 매주 1회 2시간씩 진행했는데, 혼자 공부하면 중간에 멈추는 경우가 있어서 강제성을 부여하기 위해 시작함

- 이제 중급 서적([파이썬을 이용한 클린 코드를 위한 테스트 주도 개발](http://www.yes24.co.kr/Product/goods/16886031))을 공부하자
